# Differential expression gene analysis

## Preface

接下来我们使用前面featurecounts的到的基因表达的count矩阵进行差异分析。差异分析的工具有跟多，最常用的有DESeq2，edgeR，limma等。每个包都有自己的特点。对于最常规的分析，我个人认为DESeq2最简单易用，下面我们就以DESeq2为例来进行差异分析。

- 下面的代码是R语言的代码，请提前安装好R和Rstudio（可以自行Google搜索安装方法，并不难）（所有的操作都在Rstudio中进行；如果你想或者进一步了解了R，vscode is also recommended cause拥有更多扩展性和功能）。简单学一下基础的r语言的语法和包的基本概念。可以一步一步的把代码复制给chatGPT，让它帮你理解每行代码的作用。
- 这里着重提醒一下，安装包有三种方法：
  - `install.packages("package_name")`
  - `BiocManager::install("package_name")`
  - install from GitHub using `devtools::install_github("username/repo")`
    这是因为包的来源不同，最稳妥的办法就是阅读包的官方文档。
- 在整个过程中你会发现针对一个问题有着各种各样的的包，你都可以尝试，甚至可以个性化一些函数以达到你想要的结果
